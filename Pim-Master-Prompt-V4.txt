## 1. Proje Genel Bakış

### 1.1 Vizyon
Frappe Framework üzerine inşa edilmiş, enterprise-grade Product Information Management sistemi. Tüm Frappe ekosistemi ile native entegrasyon.

### 1.2 Temel Prensipler
- **Single Source of Truth**: Golden Record pattern ile veri bütünlüğü
- **Multi-Tenant Ready**: Site bazlı izolasyon
- **Channel Agnostic**: Omnichannel içerik dağıtımı
- **AI-First**: Enrichment ve classification için AI entegrasyonu
- **Compliance Ready**: GS1, eCl@ss, UNSPSC standartları

## 4. ERPNext Entegrasyonu

### 4.1 Bidirectional Sync

```python
# hooks.py
doc_events = {
    "Product Master": {
        "after_insert": "frappe_pim.integrations.erpnext.sync_to_item",
        "on_update": "frappe_pim.integrations.erpnext.sync_to_item",
    },
    "Item": {
        "after_insert": "frappe_pim.integrations.erpnext.sync_from_item",
        "on_update": "frappe_pim.integrations.erpnext.sync_from_item",
    }
}

# Field Mapping
ERPNEXT_FIELD_MAP = {
    "Product Master.sku": "Item.item_code",
    "Product Master.product_name": "Item.item_name",
    "Product Master.short_description": "Item.description",
    "Product Master.brand": "Item.brand",
    "Product Master.default_uom": "Item.stock_uom",
    "Product Master.item_group": "Item.item_group",
    "Product Master.is_stock_item": "Item.is_stock_item",
    "Product Master.valuation_rate": "Item.valuation_rate",
}
```

### 4.2 Conditional Field Display

```javascript
// product_master.js
frappe.ui.form.on('Product Master', {
    refresh: function(frm) {
        // ERPNext alanlarını sadece entegrasyon aktifse göster
        frappe.db.get_single_value('PIM Settings', 'enable_erpnext_sync')
            .then(enabled => {
                frm.toggle_display(['item', 'item_group', 'valuation_rate'], enabled);
            });
    }
});
```
